# Examples Directory

This directory contains usage examples for Serperior API.

## Files

### 1. `test_api.py` - Quick API Test
Simple script to test if API is working.

**Run:**
```bash
python examples/test_api.py
```

### 2. `usage_examples.py` - Comprehensive Cookbook
8 real-world examples covering:
- Basic single day crawl
- Multi-day crawl
- Multiple fields comparison
- Save to CSV/JSON
- Error handling
- Batch processing
- Data analysis

**Run:**
```bash
python examples/usage_examples.py
```

Then choose from interactive menu.

### 3. `dashboard.html` - Web Interface
Visual interface for non-programmers.

**Open:**
```bash
open examples/dashboard.html
```

## Quick Start

1. Start API server:
```bash
python run_api.py
```

2. Run any example:
```bash
python examples/usage_examples.py
```
